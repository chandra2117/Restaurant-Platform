## Project Description

The Restaurant Website is a clean and elegant single-page website created to showcase restaurant details, menu items, and contact information.

The website uses smooth scroll behavior for navigation between sections like Home, About, Menu, and Contact. This project focuses on structured HTML layout and attractive CSS styling without using JavaScript or backend technologies.

- About Restaurant Section
- Menu Section
- Contact Section
- Modern UI Design
